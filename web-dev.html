Job description

A Full-Stack Web Developer creates and maintains websites. Everything from the design, layout, content, security and functionality of a web page is handled by Full-Stack Web Developers. It is worth noting that Front-End developers work on the design layout and content while Back-End developers work on security and functionality

Education/Skill Advancement

College diploma in a related field.

    Technical skills

  • PHP

  • Problem-solving
  • Teamwork

A day in the life of the career

Meeting with potential clients/employer, and discussing possible solutions with a senior developer, writing and testing code.
