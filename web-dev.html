Job description

Job description A Full-Stack Web Developer creates and maintains websites. Everything from the design, layout, content, security, and functionality of a web page is handled by Full-Stack Web Developers. It is worth noting that Front-End developers work on the design layout and content, while Back-End developers work on security and functionality.

Education/Skill Advancement

College degree or diploma in web development or design.

Skills required

    Technical skills

  • HTML
  • CSS
  • JAVASCRIPT
  • PHP

    Soft Skills

  • Communication
  • Problem-solving
  • Teamwork

A day in the life of the career

Meeting with potential clients/employers, and discussing possible solutions with a senior developer, writing and testing code, collaborating with other developers and using certain development tools, analyzing the user's needs to ensure a positive experience, you may also need to deploy hotfixes to webpages to fix issues that may affect the usability of the web page.
